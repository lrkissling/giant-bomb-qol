This extension uses the Giant Bomb API for certain features, which requires the use of an API key. You can find your key by logging in to Giant Bomb and navigating to giantbomb.com/api. Your key will never be logged or shared by Giant Bomb QoL.

Notifies you via the toolbar button when Giant Bomb is live streaming. Because of this feature's frequent API usage, you must provide your own API key above.

This hides the length of This Is The Run & Going Rogue videos, as well as the progress scrubber. When using this, you cannot jump to different parts of the video.
